# ece454

ECE 454 notes

To see slides, do

```bash
python -m http.server 8080
```

Then you can view it on `http://localhost:8080/`

You can change the following code in the `index.html` to see slides for different
weeks.

```bash
var slideshow = remark.create({
    sourceUrl: 'week3.md'
});
```
